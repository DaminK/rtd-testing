ci_lib.decomposition.decomposition.anatomical_parcellation(DecompDataObject, atlas_path=None, ROI=[], logger=LOGGER)¶

Decomposes a DecompDataObject into a anatomical parcellation based on a Brain Atlas

Parameters

  • DecompDataObject (DecompDataObject) – DecompDataObject with abitrary parcellation (Usually SVD)

  • atlas_path (String or pathlib.Path or None) – Path to Dict containing the Brain Atlas (TODO Specify Format)

  • ROI ([Strings] or None) – Regions of interest to use

  • logger (LOGGER) – The LOGGER object that all console outputs are piped into

Returns

Anatomically parcellated DecompDataObject with Spatials corresponding to the given Atlas.

Return type

DecompDataObject
