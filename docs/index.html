WIPAR¶

WIPAR (Widefield Imaging Pipeline for Analysis and Regression) is a data pipeline for processing and analysing task-specific (widefield) calcium imaging data through neural decoding. Here, calcium activity is a proxy for neuronal activations. It provides stand-alone functionalities to visualize the data analysis as well as enabling the export of processed data for other visualization purposes.

Feature Overview¶

  • Brain Alignment of different subjects and experiments

  • Different data-driven & anatomical parcellations (Atlas-based, locaNMF, SVD, ICA)

  • Different brain connectivity measurements (FC, EC)

  • Neural Decoding through simple machine learning models

  • Biomarker Detection through Feature Selection

  • Generic & customizable selection of trials and conditions

  • Interactive & automated visualization of results

  • Web-interface to review pipeline runs and export results
